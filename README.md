# ekg-vis
Ссылка на данные ЭКГ: https://disk.yandex.ru/d/q3q5UiabvBcs9w
Разархивировать в корне проекта

Для запуска нужен Node.js на компьютере 

Запуск:

1. Открыть проект в VS Code;
2. В терминале: npm i;
3. Как установится - в терминале: node server.js. Появится сообщение Example app listening on port 3000!
4. Открываешь index.html напрямую или с помощью расширения Live Server в VS Code
